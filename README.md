# 🍎 Умаревска Ябълка

Официален уебсайт на **Умаревска Ябълка** — българска фирма от село Умаревци, специализирана в отглеждането на висококачествени ябълки и производството на 100% натурален ябълков сок.

## ✨ Функционалности

- **Видео hero секция** с името на фирмата
- **Секция „Нашата история"** с второ видео и информация за фирмата
- **Сортове ябълки** — Фуджи, Пинова, Грени Смит, Златна превъзходна и Флорина, всеки с възможност за поръчка (5 / 10 / 20 кг)
- **Магазин** с работеща кошница (запазва се в браузъра):
  - Микс от ябълки — 2 €/кг (5, 10, 20 кг)
  - 100% натурален ябълков сок — 10 € / бр.
- **Блог** с 6 статии за сортове, здраве, продукти, кулинария, съхранение и фермата
- **Адаптивен дизайн** за компютър, таблет и телефон

## 🚀 Стартиране локално

Сайтът е статичен (HTML/CSS/JS), без зависимости. Просто отворете `index.html` в браузър, или стартирайте локален сървър:

```bash
python -m http.server 8000
```

После отворете `http://localhost:8000`.

## 📁 Структура

```
├── index.html        # Структура на сайта
├── styles.css        # Дизайн
├── script.js         # Кошница, блог, анимации
├── img/              # Снимки (лого, продукти, сортове, блог)
└── *.mp4             # Видео файлове за hero и история
```

## 📝 Бележки

- Контактните данни в секция „Контакти" са примерни — заменете с реалните.
- Снимките на продуктите могат да се заменят със собствени, като се запазят същите имена във `img/`.
